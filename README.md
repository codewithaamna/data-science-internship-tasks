# Data Science Internship Tasks

This repository contains the projects completed as part of my **Data Science Internship**, focusing on data analysis, visualization, and machine learning model development using real-world datasets.

---

## 📌 Internship Objectives

The main goal of these tasks was to build practical skills in:

* Data loading and inspection
* Data visualization and analysis
* Data preprocessing and cleaning
* Machine learning classification
* Model evaluation and interpretation

---

## 🧠 Task 1 — Exploring and Visualizing a Simple Dataset (Iris Dataset)

### Objective

To understand how to read, summarize, and visualize a dataset using Python.

### Approach

* Loaded the Iris dataset using **pandas**
* Explored dataset structure using `.shape`, `.columns`, and `.head()`
* Performed basic data inspection
* Created visualizations using **matplotlib** and **seaborn**

### Visualizations Created

* Scatter plots to analyze relationships between features
* Histograms to examine data distribution
* Box plots to detect outliers and value spread

### Skills Applied

* Data loading and inspection
* Data summarization
* Data visualization

---

## 💳 Task 2 — Credit Risk Prediction (Loan Prediction Dataset)

### Objective

To predict whether a loan applicant is likely to default or receive loan approval.

### Approach

* Cleaned dataset and handled missing values
* Performed Exploratory Data Analysis (EDA)
* Visualized important features such as loan amount, education, and applicant income
* Trained classification models:

  * Logistic Regression
  * Decision Tree Classifier
* Evaluated models using accuracy score and confusion matrix

### Skills Applied

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Binary classification modeling
* Model evaluation techniques

---

## 🏦 Task 3 — Customer Churn Prediction (Bank Customers)

### Objective

To identify customers who are likely to leave the bank.

### Approach

* Cleaned and prepared customer dataset
* Encoded categorical variables such as geography and gender
* Built supervised classification models
* Analyzed feature importance to understand key churn factors

### Skills Applied

* Categorical data encoding (Label Encoding / One-Hot Encoding)
* Supervised machine learning
* Feature importance analysis
* Predictive modeling

---

## 📊 Results & Insights

* Data preprocessing significantly improved model performance.
* Customer behavior and financial attributes strongly influence prediction outcomes.
* Feature importance analysis helped identify the most impactful variables affecting loan approval and customer churn.
* Visualization techniques provided clear understanding of dataset patterns and distributions.
